이미 생성한 Branch 의 이름을 변경하는 방법

```bash
git branch -m 변경전_branch_name 새로운_branch_name
```

- 굳이 삭제하고 새로 만들지 않아도 위의 명령어를 사용하면 된다.

- 그리고 브랜치를 만들 땐 develop 브랜치에서 생성하도록 습관을 들이자.
